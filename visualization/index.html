Live Dataflow Visualization (Conceptual)

This section would visualize real-time data flow and system events if connected to a live backend.

To enable live dataflow:

  1. Implement a Rust backend server that exposes the MCP API endpoints (e.g., using warp or axum).
  2. Implement WebSocket endpoints in the Rust backend to stream real-time events (e.g., entity insertions, query executions, performance metrics).
  3. Modify this frontend to connect to the WebSocket and update the visualizations dynamically.
